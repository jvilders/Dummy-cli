============
Installation
============

At the command line::

    $ pip install dummyclicli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv dummyclicli
    $ pip install dummyclicli

Or, if you are using pipenv::

    $ pipenv install dummyclicli

Or, if you are using pipx::

    $ pipx install dummyclicli
