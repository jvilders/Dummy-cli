============
Installation
============

At the command line::

    $ pip install dummy-clicli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv dummy-clicli
    $ pip install dummy-clicli

Or, if you are using pipenv::

    $ pipenv install dummy-clicli

Or, if you are using pipx::

    $ pipx install dummy-clicli
